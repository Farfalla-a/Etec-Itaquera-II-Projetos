# 🎓Portfólio de Projetos Etec — Desenvolvimento de Sistemas
## 📖 Sobre este Repositório

Bem-vindo ao meu repositório central de projetos desenvolvidos durante o curso técnico em Desenvolvimento de Sistemas na Etec. Este espaço funciona como meu portfólio acadêmico, onde organizo e demonstro a evolução do meu aprendizado prático.


## 🤔 Por que estou fazendo isso? (Contexto Acadêmico)

Este repositório é fundamental para minha formação técnica por três motivos principais:

  🟢Aplicação Prática (Mão na Massa): É aqui que eu transformo a teoria aprendida em sala de aula (como a disciplina de Linguagem de Programação) em código funcional.

  🟢Portfólio para o Futuro: Ter projetos bem documentados no GitHub demonstra proatividade e competência para futuros processos seletivos e estágios na área de T.I.

  🟢Organização e Controle: Serve como um histórico organizado do meu progresso, facilitando a revisão de conceitos e a reutilização de códigos em trabalhos futuros e no TCC.
